# PromoPing Bot Suporter

> **Note:** This is the administrative Discord bot of the PromoPing ecosystem, developed in Java using JDA (Java Discord API).

PromoPing Bot Suporter is the administrative component of the PromoPing system, a price monitoring system focused on the Portuguese market. This bot provides administrative and support functionalities through Discord, allowing you to manage and monitor the PromoPing system directly from the Discord server.

PromoPing allows users to track products across various online shops and receive automatic notifications when prices reach defined targets. The administrative bot facilitates the management and support of this system through Discord commands.

> **Note:** PromoPing monitors products chosen by the user and does not compare prices between shops.

## Architecture

PromoPing Bot Suporter was developed using the following technologies:
- **Java 11+** - Main programming language
- **JDA (Java Discord API)** - Library for Discord interaction
- **Maven/Gradle** - Dependency management

## Security

The bot implements multiple layers of protection:
- Permission validation for administrative commands
- Command input sanitization
- Structured logging and complete auditing
- Protection of tokens and sensitive credentials
- Proper exception handling

Security-related matters should be disclosed privately by contacting <corporation.promoping@gmail.com> and to report vulnerabilities, please consult [SECURITY.md](SECURITY.md).
